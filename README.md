## 📄 How to Use the Tool

The software interface remains simple. You do not need technical knowledge to perform translations.

1. Open the application.
2. Select your source document. You can drag and drop your file into the main window or use the file picker button.
3. Choose the source language from the list.
4. Press the Translate button.
5. Wait for the progress bar to complete.
6. Open your folder to find the new document. The app saves the file in the same location as your original, with a new name to identify the English version.

## 🔍 Frequently Asked Questions

**Does this work with scanned PDF files?**

This software operates on Microsoft Word (.docx) files. To translate a scanned PDF, first convert it to a Word document using your preferred scanner software or an online conversion tool.

**How does this compare to other tools?**

General translation tools often miss the nuance of legal wording. This application uses specific dictionaries for contracts and agreements to ensure the language remains accurate and binding.

**Can I translate multiple files at once?**

Yes. The software allows you to queue multiple files in the drag-and-drop window. It processes them one by one.

**Does the software change the layout?**

No. The engine maps the text to the document structure. The software makes every effort to keep your document looking exactly like the original.

**Where can I find the dictionary list?**

The application displays the supported languages in the drop-down menu on the main screen. These include major European and Asian languages typical in cross-border legal work.

## 💡 Best Practices

* **Clean Files:** Remove unnecessary graphics or complex macros from your document before starting the translation. A clean document translates faster and more accurately.
* **Review Output:** While the software follows high standards, always review the resulting text. Legal documents often carry large responsibilities. Use the built-in track changes to identify edited areas quickly.
* **Internet Connection:** Stable internet helps the translation engine access the latest definitions. If you perform very large translations, ensure your connection remains stable to avoid timeouts.
* **Naming:** Use clear names for your output files to distinguish the original language version from the English draft.
